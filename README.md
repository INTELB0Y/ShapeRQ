### Структура проекта
```
shape-rq/
├── src/
│   ├── core/
│   │   └── config.ts          // глобальный конфиг
│   ├── http/
│   │   ├── request.ts         // базовые методы get/post/put/delete
│   │   └── types.ts           // типы запросов и конфигов
│   ├── utils/
│   │   └── logger.ts          // логгер для debug
│   ├── search/
│   │   └── index.ts           // client/server search
│   └── index.ts     
├── package.json
└── README.md
```
